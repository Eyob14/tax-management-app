# tax-management-system

### This is a Minimum Viable Product (MVP) for a tax management system designed to streamline and improve tax collection processes. The motivation behind creating this system stems from the challenges faced by taxpayers in my country, where the existing tax collection system is marred by corruption and often results in significant delays for individuals completing their tax payments.

## It has the following functionalities

- Have role based authorization
- Have role based authorization
- Admin login credentials: username: admin and password: admin
- You can sign up to create a new account

## Project Setup

```sh

```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Run Unit Tests with [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```

### Run End-to-End Tests with [Playwright](https://playwright.dev)

```sh
# Install browsers for the first run
npx playwright install

# When testing on CI, must build the project first
npm run build

# Runs the end-to-end tests
npm run test:e2e
# Runs the tests only on Chromium
npm run test:e2e -- --project=chromium
# Runs the tests of a specific file
npm run test:e2e -- tests/example.spec.ts
# Runs the tests in debug mode
npm run test:e2e -- --debug
```
